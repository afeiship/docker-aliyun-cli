# webkit-sassui-linear-bg
> Webkit sass ui module for linear bg.

## demo:
+ https://afeiship.github.io/webkit-sassui-linear-bg/

## usage:
```html
<div class="box">
  <img src="test.jpg" alt="">
  <footer class="footer webkit-sassui-linear-bg" data-color=black data-deg=0>
    吉布鲁单人自助晚餐
  </footer>
</div>
```
